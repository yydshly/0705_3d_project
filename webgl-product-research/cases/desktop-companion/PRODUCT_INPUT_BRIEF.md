# Product Input Brief: Desktop Companion

## 1. What Are We Showing?

```text
Project / product name: Desktop Companion
What it is: A desktop AI companion that lives beside the user's work, reacts to context, remembers long-term preferences, and offers warm interaction.
Who it is for: People who want a more personal AI interface than a chat box, especially creators, developers, students, and users who like persistent desktop companions.
Why it matters: It turns AI from a temporary tool into a visible, ambient, emotionally warmer presence.
```

## 2. Core Capabilities Or Selling Points

```text
1. Real-time conversation and response
2. Emotional feedback through expression, motion, and ambient state
3. Long-term memory represented as persistent fragments
4. Desktop interaction with windows, reminders, files, and user activity
5. Character customization and growth over time
```

## 3. Story Direction

Preferred shape:

```text
character appears
  -> desktop environment establishes mood
  -> interaction examples
  -> memory and emotion
  -> customization and growth
  -> final hero moment
```

Opening moment:

```text
A quiet desktop scene appears. A small companion presence wakes up beside the user's workspace.
```

Middle reveal:

```text
The companion responds to chat, reacts emotionally, remembers past context, and interacts with desktop windows.
```

Final moment:

```text
The desktop becomes a calm personal workspace, with the companion, memories, and customization layers forming a warm hero view.
```
