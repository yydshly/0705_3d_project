# Publishing Guide

This prototype is a publishable research-portfolio package, not only a local movement demo.

## Preview

```bash
npm run build
npm run serve:dist
```

Open:

```text
http://127.0.0.1:5178/
```

## Film Mode

Use `Film Mode` in the top-right controls.

The mode follows this story:

1. Opening: `3D Capability Atlas`
2. Shader grass system
3. Scroll-driven 3D website
4. Spatial portfolio
5. Product film workflow
6. Reusable skill workflow
7. Closing: from research to reusable product assets

This can be screen-recorded as a short share video.

## Cover Assets

Node covers live in:

```text
public/assets/covers/
```

Current assets:

- `grass-system.svg`
- `scroll-website.svg`
- `spatial-portfolio.svg`
- `product-film.svg`
- `reusable-skill.svg`
- `showroom-poster.svg`

Replace these files with real screenshots or generated images to turn the research prototype into a stronger portfolio page.

## Suggested Recording Package

For sharing, prepare:

- One screen recording of Film Mode.
- One cover image from `public/assets/covers/showroom-poster.svg`.
- A short caption explaining that this is a WebGL / Three.js research portfolio prototype.
- A link to the GitHub repository.
